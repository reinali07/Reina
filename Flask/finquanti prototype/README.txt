website is functional offline atm

in order to run:
1. extract files
3. $ venv\Scripts\activate
4. $ set FLASK_APP=app.py
5. $ flask run

on the "charts" page, can upload data.json (or any .json data file in the same format) to produce candlestick chart

when done,
$ deactivate

disclaimer: may have bugs
website files has not been tested with the above instructions.
it was tested using the virtual environment it was created in
